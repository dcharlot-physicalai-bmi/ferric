A ternary LLM, running in this browser tab.

PrismML Ternary Bonsai 1.7B — every weight is −1, 0, or +1 — executing on your GPU via WebGPU. The exact same pure-Rust code (Ferric) runs this natively and here in WASM. Nothing is sent to a server; the model downloads once and inference is local.

Click Generate — first run downloads the ~442 MB model (cached after).
Model: prism-ml/Ternary-Bonsai-1.7B · Engine: Ferric (pure Rust, one codebase native + browser) · Q2_0 ternary weights stay packed on the GPU at 2.125 bits/weight.
